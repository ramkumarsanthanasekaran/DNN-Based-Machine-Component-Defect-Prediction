Deep Neural Network (DNN) model to predict defects in machine components using image data.

🔍 Project Overview:
• Classified components as OK vs Defective using image inputs
• Designed a DNN model architecture for accurate prediction
• Implemented a complete pipeline from preprocessing to prediction
• Evaluated performance using training accuracy and loss metrics

🧠 Model Insight:
The model learns visual patterns from casting images and predicts defects with high consistency, demonstrating the effectiveness of deep learning in industrial inspection.

📊 Key Results:
• Strong training and validation performance
• Stable convergence across epochs
• Reliable defect classification capability

🛠️ Tech Stack:
Python | TensorFlow/Keras | NumPy | Matplotlib

💡 Impact:
This work shows how AI can automate quality inspection, reduce manual effort, and improve manufacturing reliability.
